## 四、快速部署步骤

### 步骤1：服务器环境配置（元利提供脚本）
```bash
# 登录服务器后执行
# 安装Nginx
sudo apt update
sudo apt install nginx -y

# 创建网站目录
sudo mkdir -p /var/www/testqingheju.site
sudo chown -R www-data:www-data /var/www/testqingheju.site

# 上传测试文件
# 将 test-site-domain/ 目录下的文件上传到服务器
```

### 步骤2：Nginx配置
创建 `/etc/nginx/sites-available/testqingheju.site`：
```nginx
server {
    listen 80;
    server_name testqingheju.site www.testqingheju.site;
    
    root /var/www/testqingheju.site;
    index test.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    # 启用gzip压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/testqingheju.site /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### 步骤3：配置SSL证书（HTTPS）
```bash
# 安装Certbot
sudo apt install certbot python3-certbot-nginx -y

# 申请证书
sudo certbot --nginx -d testqingheju.site -d www.testqingheju.site

# 证书会自动配置，完成后访问 https://testqingheju.site
```

### 步骤4：DNS解析配置
在域名注册商控制台（腾讯云）配置：
```
记录类型：A
主机记录：@
记录值：[你的香港服务器IP地址]
TTL：600

记录类型：A
主机记录：www
记录值：[你的香港服务器IP地址]
TTL：600
```
